# SpringBootWithExposed

A simple Kotlin project to test ORM frameworks [Expose](https://github.com/JetBrains/Exposed/wiki/DAO) and [Ktorm](https://www.ktorm.org/en/schema-definition.html).
Based on Spring Boot.
